For my senior project in Computer Science, I am creating a grid-based Strategy Roleplaying Game. It is currently unfinished, but if you'd like to check it out you can head to the Github page.

npcworld

npcworld was a group project I worked on for my Machine Learning class. It simulates organisms in an environment that eat, sleep, and breed, creating new organisms by means of a genetic algorithm. The project page is here.

3D Game of Life

This is a 3D implementation of Conway's Game of Life written in Java using OpenGL. Each cell is a cube in the grid, and is colored based on how many steps through the game it has survived.

Spiral of Death

This was a 2D game I wrote in high school with Brian Mock. It was written in Python using Pygame. It is a 2-player game in which player shoot projectiles into a spiral of enemies. The player who shoots the most enemies wins the game. The projectiles had gravity applied to them. We created the graphics for the game ourselves in Inkscape.
