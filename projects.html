Here you can check out some of the more interesting projects I have worked on as a computer science student.

Pycross (working title)

This is an open source clone of the puzzle game Picross, written in Python using Pyglet. It is currently in the early stages of development, but can be found on github here.

3D Game of Life

This is a 3D implementation of Conway's Game of Life written in Java using OpenGL. Each cell is a cube in the grid, and is colored based on how many steps through the game it has survived.

Spiral of Death

This was a 2D game I wrote in high school with Brian Mock. It was written in Python using Pygame. It is a 2-player game in which player shoot projectiles into a spiral of enemies. The player who shoots the most enemies wins the game. The projectiles had gravity applied to them. We created the graphics for the game ourselves in Inkscape.

Python Tutorial

As a project during my senior year of high school, I took up learning Python and wrote a tutorial for other students to use. I also taught a workshop at my school's technology conference on coding in Python. You can download the tutorial here.
